# Privacy Policy - Brainrot Translator

**Last updated:** February 2026

## Data Collection

Brainrot Translator does **not** collect, store, or transmit any personal data. We have no servers, no analytics, and no tracking.

## What the extension accesses

- **Webpage text:** When you click "Activate", the extension reads visible text on the current page. This text is sent to the AI API provider you selected (OpenAI, Anthropic, or Groq) for translation. It is not sent anywhere else.
- **API keys:** Your API key is stored locally in your browser using Chrome's `chrome.storage.local` API. It is never transmitted to anyone other than your chosen API provider.

## Third-party services

The extension communicates only with the API provider you configure:

- OpenAI (https://api.openai.com) - [Privacy Policy](https://openai.com/privacy)
- Anthropic (https://api.anthropic.com) - [Privacy Policy](https://www.anthropic.com/privacy)
- Groq (https://api.groq.com) - [Privacy Policy](https://groq.com/privacy-policy)

Your use of these services is governed by their respective privacy policies and terms.
